## Create an issue

The title of the issue should be include the game number and your score, example:

```
Wordle 210 3/6
```

The body of the issue should contain the emoji board, examples:

```
游릴拘拘拘拘
拘拘游릳游릴游릳
游릴游릴游릴游릴游릴
```

or

```
Wordle 210 3/6
游릴拘拘拘拘
拘拘游릳游릴游릳
游릴游릴游릴游릴游릴
```
